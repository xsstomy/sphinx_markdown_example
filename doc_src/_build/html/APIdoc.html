sample 模块¶

牛顿法扩展都n次方根 \(\sqrt[k] a\) .的表达式为: \(x_{n+1} = x_n - \frac {x_n^k-a} {kx_n^{k-1}} = \frac {k-1}{k} x_n + \frac {a}{kx_n^{k-1}}\)

sample.sqrt_nt(n, m, round_=5)[源代码]¶

用法: \(sqrt\_nt(n,m,round) = \sqrt[m] n\)

参数:
  • n (int,float,long) –
    • 被开方的数
  • m (int,float,long) –
    • 开多少次方
  • round (int) –
    • 精度
返回:

开方结果
返回类型:

int,float,long
Raises:
  • TypeError - 参数类型不对时报错
  • ValueError - 被开方参数为复数时报错
 
>>> sqrt_nt(2,2,round_=2)
1.41
