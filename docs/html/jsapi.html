class ImageFile(url)¶
Parameters:
url (string): 我
A container for an image file.
>>> var img = new ImageFile( "_static/jdoctest.png" );
>>> img.url;
'_static/jdoctest.png'
fetchData()¶

Request to the server to get data of this image file. When the request done we can get size or modified attribute.

>>> img.fetchData();
>>> wait(function() { return img.data; });
>>> img.size;
21618
>>> img.modified; //doctest: +SKIP
Sat Sep 25 2010 19:57:47 GMT+0900 (KST)
